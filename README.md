This project is part of The Odin Projectâ€™s Foundations Course (more details about this project can be found [here](https://www.theodinproject.com/lessons/foundations-recipes)). It is a simple multi-page recipe website built to practice core HTML structure and Git/GitHub workflow.

## What This Project Includes

- A main index page linking to multiple recipes
- Individual recipe pages with:
    - Title
    - Dish image
    - Description
    - Ingredients list
    - Step-by-step instructions
    - Link back to the home page

Organized folder structure using a ```recipes/``` directory
